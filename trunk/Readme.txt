*** GWT and MVC togehter at last! ***

The example show how toretrofit a GWT application with MVC.

You need to have Maven (http://maven.apache.org) installed in order to build and run the application.

To run the application, execute "mvn jetty:run-war" command than hit "http://localhost:8080" URL.

All the infrastructure classes are in "example.client.core" package.
